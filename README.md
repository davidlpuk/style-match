## Getting Started

### Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Deploy to Firebase (View on Mobile)

1. Update `.firebaserc` with your Firebase project ID
2. Run deployment:
   ```bash
   npm run deploy
   ```
3. Access the URL on your mobile device!
